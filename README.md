# 🔍 小红书一键搜索

> 选中网页文字，右键或快捷键自动分屏打开小红书搜索，不跳转、不复制、不打断浏览。

## ✨ 功能特性

| 功能 | 说明 |
|------|------|
| 🖱️ **右键搜索** | 选中文字 → 右键 →「在小红书搜索」→ 自动分屏打开 |
| ⌨️ **快捷键搜索** | `⌥ Option + ⇧ Shift + ⏎ Return`，支持 REDoc 等屏蔽右键的页面 |
| 🖥️ **自动分屏** | 当前窗口缩至左半边，搜索结果新窗口开在右半边 |
| 🔄 **关闭恢复** | 关闭搜索窗口后，原窗口自动恢复原始尺寸 |
| 📋 **搜索历史** | 点击扩展图标查看最近 20 条搜索记录，一键重搜 |
| 💬 **Toast 反馈** | 搜索触发时页面顶部弹出提示 |
| 🔍 **Popup 搜索框** | 点击图标可直接输入关键词搜索，不依赖选中文字 |

## 📦 安装

1. 下载 ZIP 并解压
2. Chrome 地址栏输入 `chrome://extensions/`
3. 右上角打开「**开发者模式**」
4. 点击「**加载已解压的扩展程序**」，选择解压后的文件夹
5. 完成 ✅

## ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| `⌥ + ⇧ + ⏎` | 选中文字后按下，自动分屏搜索小红书 |

> 💡 首次安装后已打开的页面需要**刷新**才能加载脚本。

## 🎯 使用场景

- **内容运营 / 创作者** — 浏览网页看到热门话题，一键搜小红书找灵感
- **品牌营销人员** — 监控品牌词、竞品词在小红书的声量
- **电商 / 选品** — 各平台看到商品名，快速查小红书种草笔记和口碑
- **重度小红书用户** — 浏览网页时看到感兴趣的事物，习惯性去小红书搜真实评价

## 📁 项目结构

```
xhs-search-extension/
├── manifest.json      # 扩展配置 (Manifest V3)
├── background.js      # Service Worker - 右键菜单、分屏、搜索历史
├── content.js         # Content Script - 快捷键监听、Toast 反馈
├── popup.html         # Popup 页面 - 搜索框 + 历史记录
├── popup.js           # Popup 逻辑
├── icons/             # 扩展图标
│   ├── icon16.png
│   ├── icon48.png
│   └── icon128.png
├── README.md
├── LICENSE
└── .gitignore
```

## 🛠️ 技术栈

- **Manifest V3** — Chrome 扩展最新规范
- **Service Worker** — 后台右键菜单、窗口管理、搜索历史
- **Content Script** — 键盘监听（捕获阶段）、Toast 注入
- **Chrome Storage API** — 本地搜索历史（不上传服务器）
- **Chrome Windows API** — 分屏窗口管理 + 关闭自动恢复

## 🔒 隐私

- ❌ 不收集任何用户数据
- ❌ 不上传浏览记录
- ❌ 不碰后台
- ✅ 搜索历史仅存在本地 `chrome.storage.local`
- ✅ 无网络请求，纯本地运行
